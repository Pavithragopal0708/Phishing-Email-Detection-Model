# Phishing-Email-Detection-Model
## Description
Machine Learning model to detect phishing emails.
## Features
- Train on phishing and legitimate emails
- Extract email features
- Detect URLs and suspicious keywords
- Classify emails as Phishing or Safe
- Display accuracy and confusion matrix

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy

## Expected Outcome
The model successfully classifies emails as Phishing or Safe with high accuracy based on textual content and URL features.
